# Tic Tac Toe (Python)

A Tic Tac Toe game built in Python with multiple versions:
- Terminal version (Player vs Player)
- GUI version using Tkinter
- GUI version with AI (coming soon)

## Features
- Interactive gameplay
- GUI built with Tkinter
- Modular Python code
- Beginner friendly project

## Technologies Used
- Python
- Tkinter

## Project Structure
tic_tac_toe/
│
├── terminal_pvp.py
├── gui_pvp.py
├── gui_vs_ai.py

## How to Run

Clone the repository:

git clone https://github.com/your-username/python-mini-projects.git

Run the GUI version:

python gui_pvp.py
